## 常见问题
**1. 第一次打开app，提示"配置异常"或"responseCode:xxx"、"responseCode:connection reset" ？**  
  
   检查网络是否能正常打开github，网络通畅后杀进程重启，如依旧提示错误，重复此步骤。  
    
**2. 打开app后，功能无法使用且无任何提示？**

   前往模块设置弹层查看配置加载状态，弹层打开时会主动获取最新配置，读取配置成功之后会保存到本地，不会再请求网络读取配置了。
